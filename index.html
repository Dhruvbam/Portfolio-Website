DTMF Decoder

The DTMF Decoder is a Python-based application designed to interpret audio signals from telephone keypads by decoding dial tones from WAV files into corresponding numeric digits. Leveraging the Fast Fourier Transform (FFT) for detailed frequency analysis, the decoder identifies the unique dual-tone frequencies associated with each key press and accurately translates them into numbers or symbols (0-9, *, #). Noise-filtering techniques are implemented to ensure clarity, allowing the decoder to operate effectively even with background noise. This project showcases my expertise in Digital Signal Processing (DSP), real-time data handling, and Python programming.

Python | Pandas | NumPy | Fast Fourier Transform | Digital Signal Processing

Course Sequencer

The Course Sequencer is a Python application designed to optimize the order in which students should take courses based on their prerequisites. Utilizing Depth-First Search (DFS) and topological sorting within a directed acyclic graph (DAG), the tool models courses as nodes and prerequisites as directed edges. By performing a topological sort, the program generates an efficient course sequence that ensures all prerequisites are met, minimizing scheduling conflicts and streamlining academic planning. This project highlights skills in algorithm design, graph theory, and applying graph traversal techniques to real-world problems.

Python | Topological Sorting | Depth-First Search

Search Engine Reliability

The Search Engine Reliability project is a Python-based application developed to evaluate the consistency of search engine rankings. By applying sorting algorithms such as Merge Sort, Quick Sort, and Insertion Sort, the tool calculates inversion counts to detect discrepancies between rankings. Higher inversion counts indicate lower reliability, providing a quantitative measure of ranking consistency across search engines. This project demonstrates expertise in algorithm design, data analysis, and the application of sorting techniques for real-world evaluation.

Python | Algorithm Analysis | Data Science
